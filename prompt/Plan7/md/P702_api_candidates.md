# P702 API Candidates for Entertainment and Visual Upgrade

## APIs integrated in Plan7
1. Official Joke API
- URL: https://official-joke-api.appspot.com/random_joke
- Purpose: Display random joke content in Fun Kitchen Corner.
- Key requirement: None.

2. Advice Slip API
- URL: https://api.adviceslip.com/advice
- Purpose: Display dynamic advice content in Fun Kitchen Corner.
- Key requirement: None.

3. OpenStreetMap Tile API (via Leaflet)
- URL pattern: https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png
- Purpose: Render interactive campus map.
- Key requirement: None.
